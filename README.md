  FUNCTION :

  Bouttons
    Shift + step button = step jump
    Fill + shift = Clear
    Copy/Paste + shift = Save
    Pause/Play + shift = Stop

   Encoder
    Tempo + push = clock divider
    Gate length + push = Number of step
    Page navigation + push = Forward/Backward/Random

   Selecter
    Part choice
